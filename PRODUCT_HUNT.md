## Short Description

---

## Long Description

Every app you use is missing something. A "save all" button on Instagram. A speed dial for YouTube. A snooze feature in Gmail that actually works the way you want. You've thought about it. You've wished for it. But you can't build it — and even if you could, you can't share it.

Naya changes that.

Describe any feature in plain language. Naya's AI generates it as a lightweight overlay that runs on top of the app you're already using — no code, no extensions to hack together, no waiting for a product team to prioritize your request.

**How it works:**

1. **Describe** — Type what you want in the command prompt. "Add a bookmark button to every YouTube video." That's it.
2. **Generate** — Naya's AI builds the feature as a deployable overlay, complete with UI and logic. Preview it live before you ship.
3. **Deploy & Share** — Push it to your browser. Publish it to the Naya marketplace. Let anyone fork it, improve it, and redeploy.

But Naya isn't just a tool — it's a community. Browse the Explore page to find features other people have built. Vote in polls to decide what the community builds next. Fork a feature you like, tweak it, and make it yours.

The vision: GitHub for experiences. A world where the features you use aren't locked behind corporate roadmaps. Where anyone can shape the software they interact with every day. Where the best ideas — not the loudest stakeholders — decide what ships.

Naya is the feature layer the internet has been missing.

## First Comment (Maker Post)

Hey everyone — I'm the maker behind Naya.

I built this because I kept running into the same wall: I'd use an app daily, notice something obvious that was missing, and realize I had zero power to change it. Extensions exist, sure, but they're rigid, siloed, and impossible for most people to create. I wanted something where you could just *say* what you wanted and have it appear.

If you've seen tools like Wabi — Naya is different in a few key ways. First, features aren't static templates; they're AI-generated from natural language and fully customizable. Second, everything is community-driven: you can fork, remix, and vote on features. Third, Naya isn't trying to replace the apps you use — it layers on top of them, invisibly.

What's coming next:
- A Chrome extension for one-click deploy to any tab
- Collaborative feature editing (real-time, multiplayer)
- An API so developers can programmatically create and distribute overlays
- Deeper community tools — bounties, challenges, leaderboards

I'd genuinely love feedback. What features would you build first? What apps are you most frustrated with? Drop a comment — I read every single one.
